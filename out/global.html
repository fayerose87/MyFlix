movieSchema

Defines the schema for the movies collection that was created in MongoDB. Genre, Director, and Actors are subdocuments which contain additional keys holding information. To add a new movie, a title and description is required and must be a string.

userSchema

Defines the schema for the users collection that was created in MongoDB. To add a new user, the username, password, and email is required and must be a string. Favorite Movies contains an array of IDs which refer to the document within the dm.movies collection
